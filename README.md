## contents

#### wenxianzhengli 

用于整理pdf文件，可以批量查找论文作者及论文中的关键词（就是不太准）

#### aes 

利用AES协议对文件进行加密/解密

#### connect 

自动检查网络连接，当出现问题时尝试登陆p.nju.edu.cn（南大网络中心）

#### tex2txt 

去除latex代码中的公式与标签，从而方便将文本提交grammarly检查

####textCompare

检查两个文本文件是否相同
